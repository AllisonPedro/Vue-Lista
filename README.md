# Vue Lista

Feito em Vue.JS 

## Instalação

Use o [Vue Cli](https://cli.vuejs.org/guide/installation.html) na sua maquina.

```bash
npm install -g @vue/cli
# OR
yarn global add @vue/cli
```

## Servidor de desenvolvimento

```bash
cd <pasta do projeto>

# Comando para rodar o serviço
npm run serve
```

## Build de produção

```bash
cd <pasta do projeto>

# Comando para rodar o serviço
npm rum build
```
